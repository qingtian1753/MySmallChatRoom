本项目是一个基于 C++11 与 Linux 网络编程实现的命令行聊天室，采用非阻塞 socket + epoll 事件驱动模型，实现多客户端并发通信。

项目支持群聊、私聊、用户改名、在线列表查询、帮助命令，并实现了应用层心跳检测与超时连接清理机制。

核心特性包括：

- 基于 epoll 的 IO 多路复用
- 非阻塞 socket + 事件驱动模型
- 自定义应用层协议（4 字节长度头）解决 TCP 粘包拆包
- 发送队列 + 写缓冲区处理部分发送问题
- 基于 ClientSession / ClientState 的连接状态管理
- 命令表分发机制（替代 if 链）
- 应用层心跳 `/ping` `/pong` + 超时连接清理
- 使用 UniqueFd 实现 fd 的 RAII 生命周期管理
